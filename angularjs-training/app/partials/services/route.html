• Links URLs to controllers and views. Use current property to access current route, or routes to get all configured routes.
  • Used to handle partial templates included into the layout template (index.html), depending on the current route (the view currently displayed by the user).
  • Routes are declared via the $routeProvider, which wires together urls, view templates and controllers.
  • Used in conjunction with the ngView directive, which includes the template for the current route in the layout.
