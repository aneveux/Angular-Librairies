• Takes the name of a required controller to be passed into linking function.
  • If no controller is found, an error is raised. You can use ? prefix to make the dependency optional.
  • Use ^ prefix to look for the controller on parent elements as well.
require: ['^parentCtrl', '?optionalCtrl'], link: function (scope, iElement, iAttrs, controllers) { // controllers[0].callMethod(...); }
