• Used any time your application needs to react to a change in the current URL or if you want to change the current URL in the browser.
  • Warning: Does not fully reload the page when browser URL is changed. To reload the page after changing the URL, use $window.location.href.
  • Use $location.path(path) to return or set path of current url.
  • Use $location.url(url) to return or set path, search and hash (e.g. /path?a=b#hash) of current url.
