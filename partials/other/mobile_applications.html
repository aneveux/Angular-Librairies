• angular-mobile.js is available on angular versions >= 1.14. It includes fastclick.js code by default and directive ngSwipe.
  • angular-mobile-nav adds transitions to the angular navigation.
  • angular-jqm is a library which provides functionalities from jquery-mobile, but with small footprint and without any other dependency required (no jQuery or jQuery-Mobile).
    Main features are:
    • Mobile directives (jqmHeader, jqmCheckbox, etc.)
    • A new routing system, with transitions.
    • Utility services ($orientation, etc.).
